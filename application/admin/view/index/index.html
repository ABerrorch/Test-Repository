使用说明

这里是本系统的使用说明，请在第一次使用本系统的时候阅读。
首先不要使用360浏览器。360浏览器对JS的支持有一些局限，可能导致一些错误

一、个人信息

个人信息、登出按钮以及个人所拥有的权限可以点击左上角的用户名处获得。

个人账号不分大小写

二、基本权限

本系统的基本权限大体上包括三个级别

1. 管理员
拥有几乎所有权限

2. 高级用户
在管理员的基础上，不能使用左侧导航栏中的“高级权限”部分
出于为了防止误操作的考虑，关闭了高级用户删除项目的权限

3. 一般用户
在高级用户的基础上，不能使用“项目创建”功能，不能作为项目管理员或者课题管理员。

三、课题项目管理

1. 课题和项目都有相应的管理员，分别称为课题管理员和项目管理员，管理员能够使用最全面的功能。

2. 除了管理员，课题和项目还包括成员。相对于管理员，成员只能使用部分功能。

3. 用户只能看见自己所属的课题和项目

详细情况请点击左侧导航栏观察。
